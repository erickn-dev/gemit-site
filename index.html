Consistent commits and branches with AI

gemit generates Conventional Commit messages, suggests standardized branch names, and asks for confirmation before running git commands.

Install View Commands

Global installation

Works on Windows, Linux, and macOS via npm.

npm i -g @ericknovaes56/gemit-cli

Initial setup

Configure provider, model, and API key with a guided wizard.

gemit init
gemit doctor

Core commands

gemit

Suggests a commit message and asks whether to commit.

gemit commit

Explicit commit suggestion flow.

gemit branch "description"

Suggests a branch name and creates it after confirmation.

gemit doctor

Validates provider, model, and API key configuration.

.env example

LLM_PROVIDER="google"
LLM_MODEL="gemini-2.5-flash"

GOOGLE_API_KEY=""
GEMINI_API_KEY=""
OPENAI_API_KEY=""
ANTHROPIC_API_KEY=""

Recommended workflow

  1. 1. Install with npm i -g @ericknovaes56/gemit-cli.
  2. 2. Run gemit init in your project.
  3. 3. Create a branch with gemit branch "description".
  4. 4. Generate commits with gemit.
  5. 5. If setup fails, run gemit doctor.
